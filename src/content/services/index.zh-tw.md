# 服務項目

行雲資訊提供涵蓋 IT 維運、資安防護、資產管理與客製開發的完整服務。每項專案皆包含架構規劃、建置導入、文件化與持續優化，協助企業專注在核心業務。

## 服務總覽
- **全面 IT / MIS 代管**：IT 監控系統、容器與 DevOps、IT 外包與辦公室網路建置。
- **機房與雲端**：IDC 建置、虛擬化與 Proxmox 叢集、網站及伺服器代管營運。
- **資安管理**：Wazuh SIEM、Zero Trust、備份與郵件安全、802.1X、VPN 及 ZITADEL IAM。
- **資產管理**：Wazuh 資產可視化與 Jamf 企業級 Apple 設備管理。
- **客製需求**：依企業流程打造的專屬系統，整合既有應用與資料流。
